Selecting Section:

Sections can be selected on a layer by typing in the section label and pressing 'Go To' or by selecting a single feature of the network layer and clicking 'From Layer'.

Selecting Chainage:

Once a section has been selected a marker will appear on the map corresponding to the chainage in the box. The 'From Click' button will cause clicking on the map to move the marker and chainage to the nearest point of the selected section.

Setting Layer and Fields:

In order to do this the layer, field with section label and field with section length will need to be set ('Layer' tab).

Layer should usually be set to the network table,label field set to sec,length field to meas_len.
