Database Setup

Connecting to Database:

Connect plugin to a database through Database,"Connect to Database..."

Setting up Existing Database for site categories

To set up an existing database for this plugin:
  • Connect to it as previously
  • Click Database,"Setup Database for site categories..." .This creates a schema named 'categorizing' that will contain everything required by the plugin.
  • Insert values into the network table.
  • Insert values into the categories table.
  • If using gradients and/or curvatures insert these into the other_events table.
