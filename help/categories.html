categorizing.categories

The categories table defines what irl should be assigned to each site category.

  • cat: site category
  • irl:investigatory reference level
  • pos: defines which event should take precedence when multiple events affect a location. The event with the lowest pos takes precedence and appears first in the output. As irl increases pos should decrease (want highest irl to take precedence).
  • description:note for benefit of people categorizing. Does not appear in results.
