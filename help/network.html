Network table

Table containing road network

- sec = section label

- geom = linestring geometry.

- meas_len = 'official' length of section. Used to calculate section chainage.

- one_way = True where section is one way.

- road = road name

- rbt = True where section is roundabout

category=default category for road. where no events road will be given this category. Should be A,B,C,R
