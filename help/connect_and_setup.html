Database Setup

Connecting to Database:

Connect plugin to a database through Database,"Connect to Database..."

Setting up Existing Database for site categories

To set up an existing database for this plugin:
  1. Connect to it as previously
  2. Click Database,"Setup Database for site categories..." .This creates a schema named 'categorizing' that will contain everything required by the plugin.
  3. Insert values into the network table.
  4. Insert values into the categories table.
  5. If using gradients and/or curvatures insert these into the other_events table.
  6. If irl depends on more than site category rewrite the irl function.
  7. Process all sections with 'select categorizing.process(sec) from categorizing.network'.
  8. Plugin is now ready to use. The plugin will update the op table when editing junctions and crossings.

The irl function:

When processing sections the database uses the irl(ct text,sect text) function to calculate irl given category ct and section label sect.

The default implementation of this function looks up the irl value for ct from the categories table.

The function will need replacing if irl depends on section as well as site category.
