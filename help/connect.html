Import the network into QGIS:

The network is in table categorizing.network. Import this into qgis.

Set fields:

The plugin requires a layer with the road network. This needs to have a field with section label and a field with section length.

Under the layer tab: set network layer to the layer with the network,label field to 'sec' and length field to 'meas_len'.

Connect:

Connect the plugin to the database through the "Connect..." button.
